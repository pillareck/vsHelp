# UI / UX Guidelines

## Estilo Visual

O sistema deve possuir:

* aparência moderna
* visual clean
* dark theme
* baixa poluição visual
* boa legibilidade
* foco em produtividade

---

## Sidebar

A sidebar deve:

* ser minimalista
* possuir hover suave
* possuir item ativo bem definido
* manter espaçamento consistente
* evitar excesso de elementos

---

## Botões

Os botões devem:

* possuir bordas arredondadas suaves
* possuir hover moderno
* evitar aparência WinForms
* possuir padding confortável
* possuir contraste adequado

Evitar:

* botões quadrados
* gradientes exagerados
* sombras pesadas

---

## Inputs

Os inputs devem:

* possuir boa área de clique
* possuir focus visual claro
* possuir placeholder discreto
* manter contraste confortável

---

## Cards

Os cards devem:

* possuir separação visual suave
* possuir padding consistente
* possuir leve profundidade
* evitar bordas exageradas

---

## Tipografia

Preferências:

* hierarquia clara
* boa legibilidade
* espaçamento confortável
* visual moderno

Fontes recomendadas:

* Segoe UI
* Inter
* Cascadia Code (logs)

---

## Logs

A área de logs deve:

* possuir visual técnico/profissional
* utilizar fonte monospace
* possuir contraste confortável
* facilitar leitura operacional

---

## Responsividade

A interface deve:

* adaptar ao redimensionamento
* evitar quebra visual
* manter proporções consistentes

---

## Evitar

* visual corporativo antigo
* excesso de bordas
* excesso de gradientes
* excesso de animações
* elementos visuais desnecessários
* aparência padrão WinForms
